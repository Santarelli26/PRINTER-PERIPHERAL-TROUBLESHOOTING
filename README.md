 # School Printer & Peripheral Troubleshooting

## Overview
This project documents common printer and peripheral issues found in school environments and their resolutions.

## Devices Covered
- Network printers
- USB printers
- Scanners
- Monitors
- Keyboards & mice

---

## Printer Troubleshooting Scenarios

### Scenario 1: Printer Offline
**Diagnosis:**
- Verified network connection
- Checked IP address
- Restarted print spooler

**Resolution:**
- Reinstalled printer driver
- Assigned correct IP

---

### Scenario 2: Driver Compatibility Issue
**Diagnosis:**
- Verified OS architecture
- Checked manufacturer driver support

**Resolution:**
- Installed correct 64-bit driver

---

### Scenario 3: Paper Jams & Error Codes
**Diagnosis:**
- Inspected rollers
- Cleared debris

**Resolution:**
- Cleaned rollers
- Reset printer
